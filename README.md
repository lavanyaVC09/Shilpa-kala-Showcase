The Shilpa-Kala Showcase
The Shilpa-Kala Showcase is a mobile application developed to bridge the digital divide for rural stone carvers and traditional artisans. The project serves as a digital gallery that preserves national heritage by allowing artisans to showcase the evolution of their masterpieces through a "Process Gallery." By integrating modern features like smart WhatsApp inquiries, the app enables direct communication between creators and global buyers, turning traditional craftsmanship into a sustainable digital business. Problem Statement Traditional rural artisans often struggle with a "digital divide" that prevents them from reaching modern markets. This project provides a digital platform to showcase their craftsmanship, helping to preserve national heritage and connect stone carvers directly with global buyers to create sustainable livelihoods.

Key Features Process Gallery: A dedicated module that allows users to see the step-by-step evolution of a masterpiece, highlighting the skill involved in traditional stone carving.

Smart WhatsApp Inquiries: A direct communication feature that lets potential buyers contact artisans instantly for customized orders.

Interactive UI: A responsive and smooth user experience designed to make traditional art accessible to digital users.

Tech Stack Language: Kotlin for robust and modern application logic.

UI Framework: Jetpack Compose used to build a declarative and high-performance user interface.

Image Loading: Glide Library integrated for efficient memory caching and smooth display of high-resolution art images.
